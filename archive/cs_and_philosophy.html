What is this thing?
- A sequence of zeroes and ones
- A giant number
- A text file
- A small image
- The encoded answer to the most important question of your life

It could be any of the above.

A turing machine is a theoretical machine which reads and writes an infinitely long strip of zeroes and ones. Going one cell at a time, following the rules it is given, the machine is able to implement any computer algorithm. Devised in the mid-20th century by mathematician Alan Turing, it has become the basis for our definition of computing.

Today, when we think of computers, we don't think in terms of zeroes and ones; we think in terms of music apps, messages on social media, and search bars. That is, we build abstractions on top of a basic binary foundation.

# The Forms and Dualism

While Plato places the most authenitc, true entities, the realities of things, which he calls the Forms, highest on the vertical line of existence, and mere shadows or replicas lowest, computer scientists do the opposite: we call the most basic fundamentals low-level and the abstractions high-level. In between, countless layers make up the way software and systems are developed and run.

For example:

1 ("highest" level): When we open up a music player and choose a song, most of us think of just that: we click on a "song", and the computer "plays it".
2 (lower level): By clicking on the button that represents a song, the music player program opens a file in our file structure, then uses a library to read the file and instruct the system to play it
3: To open the file, the music player's code calls the application programming interface (API) for a filesystem library
4: The filesystem library uses system calls to directly access the file
5: Machine code is executed
6 ("lowest" level): The binary data stored in the computer is changing
